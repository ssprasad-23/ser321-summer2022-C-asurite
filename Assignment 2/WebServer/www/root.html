You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /agecalculator Example: agecalculator?month=01&year=2000 --return the present age
  • /charactercount Example: charactercount?string=abcd&char=a --return the count of input character
